# TX Packages

Official package repository for TX Terminal.

## Vision

TX Packages is a modern Android-first package ecosystem designed for TX Terminal.

## Features

- Native TX packages (.txpkg)
- AArch64 first
- Lightweight
- Open source
- Community driven

## Repository Structure

packages/    Package definitions
repo/        Binary repository
scripts/     Build scripts
tools/       Development tools
docs/        Documentation
